# 项目设计文档：自适应光谱追踪渲染器

## 一、项目概述

### 1.1 项目定位
基于 Vulkan 1.4 + Ray Query 实现的**物理正确的色散渲染器**，采用自适应光谱分裂策略，利用 RTX 4060 及以上硬件的 RT Core 加速，实现实时/交互级性能。语言标准C++20，CMake版本4.3.1

### 1.2 核心特性
- 连续光谱建模（380nm-780nm）
- 自适应波长区间分裂
- 色散向量传播（支持多次反射/折射）
- GPU 光线重排序优化
- 硬件加速求交（RT Core）

## 三、核心数据结构

### 3.1 SpectralRay

```cpp
struct SpectralRay {
    // === 几何信息 ===
    vec3 origin;           // 当前位置
    vec3 direction;        // 传播方向（归一化）
    
    // === 色散信息 ===
    vec3 dispersion;       // 色散向量 = 单位距离内紫边相对于中心的横向偏移
                           // 垂直于 direction，长度 = |dir_violet - dir_center|
    
    // === 光谱信息 ===
    int lambda_start;      // 波长区间起点 (nm, 380-780, 整数)
    int lambda_end;        // 波长区间终点 (闭区间)
    
    // === 物理量 ===
    vec3 energy;           // RGB 能量权重（被材质吸收后衰减）
    vec3 last_split;       // 上次分裂/表面交互时记录的横向分离向量
                           // 总横向分离 = last_split + dispersion * distance
    
    // === 追踪控制 ===
    int bounce;            // 剩余弹射次数
    int generation;        // 分裂代数（限制最大深度）
};
```

### 3.2 Material

```cpp
struct Material {
    int type;              // DIELECTRIC, METAL, LAMBERTIAN
    
    // 色散参数（柯西公式: n = A + B/λ²）
    vec3 cauchy_A;         // A 系数 (RGB)
    vec3 cauchy_B;         // B 系数 (RGB)
    
    // 光学属性
    vec3 albedo;           // 基础颜色/反射率
    vec3 absorption;       // 吸收系数（比尔-朗伯定律）
    float roughness;       // 粗糙度（用于模糊反射）
    float ior;             // 基准折射率（用于无色散近似）
};
```

### 3.3 RayAction

```cpp
enum RayAction {
    TERMINATE,   // 能量耗尽或超出弹射次数
    SPLIT,       // 需要分裂
    REFRACT,     // 折射
    REFLECT,     // 反射
    MISS         // 未命中，累加背景色
};
```

### 4.2 分裂决策

```glsl
bool shouldSplit(SpectralRay ray, float distance_to_next_event) {
    // 预测到达下一个事件时的总横向分离
    vec3 predicted_sep = ray.last_split + ray.dispersion * distance_to_next_event;
    float predicted_magnitude = length(predicted_sep);
    
    // 条件1：预测分离超过像素阈值
    float pixel_threshold = getPixelSizeAtDistance(distance_to_next_event);
    if (predicted_magnitude > pixel_threshold) return true;
    
    // 条件2：波长区间足够宽（避免无限分裂）
    if (ray.lambda_end - ray.lambda_start < 10) return false;
    
    // 条件3：能量足够（低能量不必分裂）
    if (length(ray.energy) < 0.05) return false;
    
    // 条件4：分裂代数限制
    if (ray.generation > 5) return false;
    
    return false;
}
```

### 4.3 分裂执行

```glsl
void splitRay(SpectralRay ray, out SpectralRay left, out SpectralRay right) {
    int lambda_mid = (ray.lambda_start + ray.lambda_end) / 2;
    float total_width = ray.lambda_end - ray.lambda_start;
    float left_width = lambda_mid - ray.lambda_start;
    float right_width = ray.lambda_end - lambda_mid;
    
    // 左子光线（紫端）
    left = ray;
    left.lambda_end = lambda_mid;
    left.energy = ray.energy * (left_width / total_width);
    left.generation = ray.generation + 1;
    // last_split 继承，不改变
    
    // 右子光线（红端）
    right = ray;
    right.lambda_start = lambda_mid;
    right.energy = ray.energy * (right_width / total_width);
    right.generation = ray.generation + 1;
}
```

### 4.4 色散向量计算（折射时）

```glsl
vec3 computeDispersion(vec3 incident, vec3 normal, 
                        float lambda_center, float lambda_violet,
                        Material mat) {
    // 中心波长方向
    vec3 dir_center = refract(incident, normal, 1.0, ior(lambda_center, mat));
    // 紫边波长方向
    vec3 dir_violet = refract(incident, normal, 1.0, ior(lambda_violet, mat));
    
    // 色散 = 紫边方向 - 中心方向
    vec3 dispersion = dir_violet - dir_center;
    // 移除沿传播方向的分量（色散是横向的）
    dispersion = dispersion - dot(dispersion, dir_center) * dir_center;
    
    return dispersion;
}
```

### 4.5 last_split 变换规则

| 事件               | 变换规则                                                     | 说明           |
| ------------------ | ------------------------------------------------------------ | -------------- |
| **传播**           | `last_split += dispersion * distance`                        | 累积横向分离   |
| **折射**           | `last_split = rotate_to_new_frame(last_split, old_dir, new_dir)` | 变换到新切平面 |
| **反射**           | `last_split = reflect(last_split, normal)`                   | 镜像反射       |
| **分裂**           | `last_split` 不变                                            | 子光线继承     |
| **进入无色散介质** | `last_split` ，`dispersion` 不变                             | 不再增加分离   |

## 五、优化策略

### 5.1 光线重排序（Sorting）

**问题**：不同光线的分裂决策不同，导致线程束发散，GPU 利用率下降。

**方案**：每轮迭代后，将所有光线按 `RayAction` 分类到不同桶。

```cpp
// 阶段1：分类
// 在 Compute Shader 中遍历所有活跃光线
RayAction action = classifyRay(ray);
uint bucket_index = getBucketIndex(action);  // 0-4
uint offset = atomicAdd(counter[bucket_index], 1);
output_indices[offset] = ray_index;

// 阶段2：重排
// 使用前缀和确定每个桶的起始位置
uint prefix[5];
prefix[0] = 0;
for (int i = 1; i < 5; i++) {
    prefix[i] = prefix[i-1] + counter[i-1];
}

// 阶段3：分桶执行
// 为每个桶单独 dispatch，每个桶内光线执行相同代码路径
dispatch(SPLIT_BUCKET, counter[SPLIT_BUCKET]);
dispatch(REFRACT_BUCKET, counter[REFRACT_BUCKET]);
// ...
```

### 5.2 其他优化

| 优化               | 方法                                    | 预期收益        |
| ------------------ | --------------------------------------- | --------------- |
| **预计算折射率表** | 波长 → n 的 LUT（纹理），避免运行时计算 | 30-50% 减少指令 |
| **自适应阈值**     | 按屏幕空间像素大小动态调整分离阈值      | 远处物体少分裂  |
| **能量截断**       | `length(energy) < 0.01` 的光线直接终止  | 减少无效光线    |
| **区间合并**       | 方向相近、波长相邻的光线可合并          | 降低光线总数    |
| **双缓冲队列**     | ping-pong 缓冲区存储每轮光线            | 避免重新分配    |
| **复用 Ray Query** | 每个线程复用一个 query 对象             | 减少寄存器压力  |

## 八、关键风险与对策

| 风险               | 影响                   | 对策                                   |
| ------------------ | ---------------------- | -------------------------------------- |
| 光线数量爆炸       | 内存溢出/性能崩溃      | 限制分裂代数(≤5) + 能量截断 + 区间合并 |

| 调试困难           | 难以定位着色器 bug     | CPU 参考实现验证核心算法               |
| 数值稳定性         | 色散向量累积误差       | 定期归一化，使用 double 中间计算       |

### B. 波长转 RGB 快速函数

```glsl
vec3 wavelengthToRGB(float lambda) {
    vec3 rgb = vec3(0.0);
         if (lambda < 440.0) rgb = vec3(0.5 - 0.5 * (lambda - 380.0) / 60.0, 0.0, 1.0);
    else if (lambda < 490.0) rgb = vec3(0.0, (lambda - 440.0) / 50.0, 1.0);
    else if (lambda < 510.0) rgb = vec3(0.0, 1.0, 1.0 - (lambda - 490.0) / 20.0);
    else if (lambda < 580.0) rgb = vec3((lambda - 510.0) / 70.0, 1.0, 0.0);
    else if (lambda < 645.0) rgb = vec3(1.0, 1.0 - (lambda - 580.0) / 65.0, 0.0);
    else if (lambda <= 780.0) rgb = vec3(1.0, 0.0, 0.0);
    
    float attenuation = 1.0;
    if (lambda < 420.0) attenuation = 0.3 + 0.7 * (lambda - 380.0) / 40.0;
    if (lambda > 700.0) attenuation = 0.3 + 0.7 * (780.0 - lambda) / 80.0;
    return rgb * attenuation;
}
```
