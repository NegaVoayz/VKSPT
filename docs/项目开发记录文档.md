2026-06-10 05:12 光谱积分重构、色散系统修复、材质系统增强

  === 光谱积分模型重构 ===
  根本问题：旧的 white-balance normalization（colorAccum /= totalWlIntegral）导致 Fresnel 反射与透射双
  重计数同一波长区间→totalWlIntegral 膨胀→归一化后图像变暗。经多轮测试（fresnelR=0, reflEnergy=0,
  反射色散清零, 反射波长为零, 交换入栈顺序等）确认：Fresnel 反射复制了入射射线的波长范围，而 normalizer
  把两份都当成新的光谱覆盖面积统计了。

  最终方案：彻底移除 white-balance normalization。新模型：
    beam 击中天空 → pixel += skyColor × throughput × wlFrac
    pixel = sum(all beams) / SPP
  无需 totalWlIntegral，无需 fromReflection，无需 wlContrib 积分。天空/环境贴图自身即光源——每束
  射线仅按波长覆盖比例 (wlFrac) 和路径衰减 (throughput) 加权贡献。

  移除内容：
  - totalWlIntegral 累积与 colorAccum /= totalWlIntegral 归一化
  - fromReflection 字段（RayState + PackedRay + 所有子射线传播）
  - NUM_REF_SAMPLES 常量 + refLambda/refWlRGB/fullSpecRef 预计算
  - estimateSpectrumFromRGB() 函数
  - wlContrib 计算与 step 函数积分

  === 能量→Throughput 重命名 ===
  energy → throughput：准确描述其物理含义——经吸收、Fresnel、俄罗斯轮盘赌后存活的入射光比例。

  2. 关闭 force-split → 无子射线→无色散（所有波长沿同一条中线折射）

2026-06-10 05:47 二阶勒让德衰减展开（Phase 7）

  核心改进：用二阶勒让德展开替代标量 throughput 表示波长相关衰减 D(λ)=exp(-τ(λ))。
  每条 beam 携带 3 个系数 (c₀, c₁, c₂) 替代单一标量，消除吸收驱动的光谱分裂。

  === 数学基础 ===
  勒让德多项式 P₀(x)=1, P₁(x)=x, P₂(x)=(3x²-1)/2 定义在归一化波长 x∈[-1,1] 上，
  其中 x(λ)=(λ-λ_mid)/Δ, λ_mid=(λ_start+λ_end)/2, Δ=(λ_end-λ_start)/2。

  衰减 D(x)=exp(-τ(x)) 的勒让德展开系数：
